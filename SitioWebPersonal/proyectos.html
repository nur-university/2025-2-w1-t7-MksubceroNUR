1. Editor de Imágenes

Un editor de imágenes con interfaz gráfica desarrollado en Java. Permite cargar, modificar y guardar imágenes, aplicando transformaciones como filtros, ajustes de brillo y contraste. Se implementaron patrones de diseño como Observer y Logger para mantener un código más modular y escalable.

2. Sistema de Gestión de Zoológico

Sistema desarrollado en Python con Tkinter y PostgreSQL para la gestión de animales, cuidadores y registros médicos. Incluye un dashboard con gráficas, un sistema de login con roles y funcionalidades CRUD completas. Se aplicó el manejo de transacciones y triggers en la base de datos.

3. Resolución del Problema de las N-Reinas

Implementación en Java y Python del clásico problema de las N-Reinas usando backtracking. La solución incluye una interfaz gráfica que muestra el tablero de ajedrez y la colocación de las reinas en tiempo real. También se desarrolló un análisis de complejidad Big-O y un informe académico.
